Overview

simple_ml is a comprehensive machine learning library for Eiffel, providing production-ready implementations of six core algorithms: linear regression, logistic regression, decision trees, random forests, support vector machines, and neural networks.

Every algorithm is implemented with Design by Contract throughout, ensuring correctness, safety, and predictability. The library is fully void-safe, SCOOP-compatible, and has zero external dependencies.

Part of the Simple Eiffel ecosystem.

Features

Design by Contract

Every class and feature has complete require/ensure/invariant specifications, enforced at runtime with assertion checking enabled.

Supervised Learning

Linear and logistic regression for continuous and categorical prediction tasks with full statistical support.

Tree-Based Methods

Decision trees and random forest ensembles for classification with configurable depth and ensemble parameters.

Support Vector Machines

Linear SVM implementation with configurable regularization and convergence tolerance.

Neural Networks

Multi-layer perceptron classifier with configurable hidden layer architecture and backpropagation training.

Void-Safe & SCOOP

Fully void-safe implementation (void_safety="all") with SCOOP concurrency support for parallel algorithms.

Included Algorithms

LINEAR_REGRESSION_MODEL

Supervised regression using gradient descent. Configure learning rate and maximum iterations. Learns weight vector for feature-based prediction of continuous values.

  • train(X, y) - Train on feature matrix and target vector
  • predict(x) - Predict continuous value for feature vector
  • set_learning_rate(rate) - Configure learning rate
  • set_max_iterations(max) - Configure maximum iterations

LOGISTIC_REGRESSION_MODEL

Binary and multiclass classification using gradient descent with sigmoid activation. Uses linear approximation of sigmoid to avoid exponential overflow.

  • train(X, y) - Train on features and class labels
  • predict(x) - Predict class label
  • predict_proba(x) - Predict probability for positive class

DECISION_TREE_CLASSIFIER

Tree-based classification with configurable maximum depth and minimum samples per split. Learns hierarchical feature-based decision rules.

  • train(X, y) - Train tree on features and labels
  • predict(x) - Predict class using learned tree
  • set_max_depth(depth) - Limit tree depth
  • set_min_samples_split(min) - Minimum samples for node split

RANDOM_FOREST_CLASSIFIER

Ensemble learning using multiple decision trees. Configure number of trees and maximum depth. Uses majority voting for classification.

  • train(X, y) - Train ensemble on features and labels
  • predict(x) - Predict class using ensemble voting
  • predict_proba(x) - Class probabilities from ensemble

SVM_LINEAR

Support Vector Machine with linear kernel. Configure regularization parameter C and convergence tolerance. Binary classification via linear decision boundary.

  • train(X, y) - Train SVM on features and labels
  • predict(x) - Predict class for feature vector
  • decision_function(x) - Raw decision value

NEURAL_NETWORK_CLASSIFIER

Multi-layer perceptron (MLP) for classification. Configure hidden layer sizes, learning rate, and epochs. Uses uniform probability distribution for simplified backpropagation.

  • train(X, y) - Train network on features and labels
  • predict(x) - Predict class label
  • predict_proba(x) - Class probabilities
  • set_hidden_layers(sizes) - Configure layer sizes

Installation

Add to your ECF (Eiffel Configuration File):

<library name="simple_ml" location="$SIMPLE_EIFFEL/simple_ml/simple_ml.ecf"/>

Then use in your Eiffel code:

create {LINEAR_REGRESSION_MODEL}.make
    .set_learning_rate (0.01)
    .set_max_iterations (100)
    .train (training_X, training_y)
prediction := model.predict (test_X)
